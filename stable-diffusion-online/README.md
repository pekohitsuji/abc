# Stable Diffusion Online あれこれ

ChatGPT の回答をまとめた文書

- 重み付けはプロンプトで指定できる
- 以下の Denoise と CFG は Stable Diffusion Online では隠蔽されている

## Denoise

- 「元画像をどれだけ壊すか」の強さ
  1. 画像をノイズ（砂嵐）に近づける
  2. プロンプトに従って再構築する
- Denoise 低い = プロンプトが効きにくい
- Denoise 高い = 元画像が崩れる
- 数値の意味 (0.0〜1.0)

| 値  | 挙動             | 備考 |
|----:|:-----------------|:-----|
| 0.1 | ほぼ元画像       | |
| 0.2 |                  | 髪型・顔ほぼ固定、色味は変わるかも |
| 0.3 | 少しだけ変化     | |
| 0.5 | かなり変わる     | 顔が変わる可能性あり、構図・体型は維持しやすい |
| 0.7 | 別人レベルになる | |
| 0.8 |                  | ポーズまで変わる、別画像に近い |
| 1.0 | 元画像ほぼ無視   | |

- 用途に合わせる

| 用途           | 推奨 Denoise |
|:---------------|-------------:|
| 顔だけ変えたい | 0.45〜0.6    |
| 衣装変更       | 0.5〜0.7     |
| ポーズ変更     | 0.6〜0.8     |
| 微修正         | 0.2〜0.4     |


## CFG

- プロンプトを どれだけ守らせるか
- CFG Scale = Classifier Free Guidance Scale

| CFG    | 挙動                |
|:-------|:--------------------|
| 3〜5   | ゆるい・自由度高い  |
| 6〜9   | バランス良い (推奨) |
| 10〜14 | 指示を強く守る      |
| 15以上 | 破綻しやすい        |

- 実際の違い (例: blue eyes)

| CFG | 結果 |
|----:|:-----|
|   5 | 青にならないこともある、生成が自然 |
|   8 | 安定して青になる、バランス良い     |
|  13 | 強烈な青や不自然な発色になることも |

- 用途に合わせる

| 用途           | CFG   |
|:---------------|:------|
| 顔変更         | 7〜9  |
| 衣装変更       | 8〜10 |
| ぬいぐるみ削除 | 6〜8  |
| ポーズ変更     | 8〜11 |


## Denoise と CFG

- 変化量 ≒ Denoise × CFG

| 指定                 | 結果             |
|:---------------------|:-----------------|
| Denoise 高 × CFG 高 | 崩壊しやすい     |
| Denoise 低 × CFG 高 | あまり変わらない |
| Denoise 中 × CFG 8  | 安定             |


## 重み付け

- どの要素がどれだけ重要か
- 1.6 以上は、顔が崩れる、不自然になる、他要素が無視される
- オススメは 1.1 〜 1.4
- (blue eyes:1.2) 青い目は大事
- (background:0.7) 背景は控えめ


## Denoise と CFG と重み付け

影響力 ≒ (重み) × (その概念が適用される空間の広さ) × (CFG) × (Denoise)

| 重み付け | 局在的圧力       | 注文内の優先順位           |

| Denoise | 意味                                     |

|     0.2 | この絵ほぼそのままで、ちょっとだけ直して |

| CFG | 意味                               |
|----:|:-----------------------------------|
|   5 | まぁ、自由にやって                 |
|  12 | この仕様 (プロンプト) を絶対に守れ |
